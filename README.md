## Features

- Shows **Last Key, KeyCode, Code**

- **Dark neon theme**
- Placeholder for **ads or donations**

## Usage
1. Open `index.html` in a browser (desktop or mobile)
2. Type in the input box
3. See the last key, key code, and history
4. Click **Clear** to reset
